How Next Run Is Calculated

nextRun = firstRun + N × intervalSeconds  where  N = ceil((now - firstRun) / intervalSeconds)

Example: firstRun = 2025-01-01 12:00, interval = 28800s (8h). If now = 2025-05-16 09:45, then N = ceil(3,822,708 / 28,800) = 133, so nextRun = 2025-01-01 12:00 + 133 × 8h = 2025-05-16 12:00.
